# Code of Conduct

## Our Commitment

We want RetailPulse to be a respectful, inclusive, and productive project space for everyone.

## Expected Behavior

- Be respectful and constructive in feedback.
- Assume good intent and communicate clearly.
- Focus on ideas, evidence, and project outcomes.
- Help keep discussions actionable and reviewable.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or hostile communication
- Sharing private or sensitive information without permission
- Any conduct that creates an unsafe environment

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this code of conduct.

## Reporting

To report behavior concerns, open a private security/disclosure issue or contact project maintainers through repository channels. Include relevant context and links so maintainers can investigate.
